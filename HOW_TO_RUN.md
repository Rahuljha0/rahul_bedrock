# How to Run RsChat_Boot_AI Web Application

## Quick Start

1. Open Command Prompt in this folder

2. Run the application:
   ```
   run_app.bat
   ```

3. Open your browser and go to:
   ```
   http://localhost:5000
   ```

## How to Use Custom Prompts

1. Click "Custom Prompt" button in Prompt Engineering section
2. Enter your custom instructions, for example:
   - "Act as a teacher and explain with examples"
   - "Answer like I'm 10 years old"
   - "Provide code examples with explanations"
   - "Be encouraging and motivational"
3. Type your question and send
4. The AI will follow your custom instructions!

## Troubleshooting

If the app doesn't start:
- Make sure port 5000 is not in use
- Check that AWS credentials are set in run_app.bat
- Ensure Flask is installed: `pip install flask`
